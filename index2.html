TowerZoom

Clean & Minimalistic Design

TowerZoom is a Project Management tool intended to better the user experience for customers and internal users alike.

A work in progress, the main function of TowerZoom is to automate tasks that currently result in lost time for the company and delays for the customer. Specifically, TowerZoom includes functionality to use existing lease terms to calculate pricing for our clients, whereas in the current workflow, all of that data needs to be calculated manually for every transaction.
